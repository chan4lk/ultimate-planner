# Ultimate Planner

Ultimate Planner is a modular monolith FastAPI backend.

## Quick Start

1. Install [uv](https://github.com/astral-sh/uv):
   ```bash
   pip install uv
   ```
2. Install dependencies:
   ```bash
   uv sync 
   ```
3. Run the app:
   ```bash
   uvicorn app.main:app --reload
   ```
